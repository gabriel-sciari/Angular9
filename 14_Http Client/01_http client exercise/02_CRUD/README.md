
# Task 

* Create this GET request:
```
    url: 'https://jsonplaceholder.typicode.com/posts/1'
    method: 'GET'
```

* Create this POST request:
```
    url: 'https://jsonplaceholder.typicode.com/posts'
    method: 'POST'
    body:{
      title: 'foo',
      body: 'bar',
      userId: 1
    }
```


* Create this PUT request:
```
    url: 'https://jsonplaceholder.typicode.com/posts/1'
    method: 'PUT'
    body:{
      id: 1,
      title: 'foo',
      body: 'bar',
      userId: 1
    }
```
  
* Create this DELETE request:
```
    url: 'https://jsonplaceholder.typicode.com/posts/1'
    method: 'DELETE'
```
